## Capstone Project

This repository is being developed as part of the FlyRank Machine Learning track.

The capstone project focuses on building a Search Intelligence system that uses anonymized search performance data to prioritize webpages for optimization.

### Current Capstone Direction

**Research question:** Can a machine learning model help identify webpages that are likely to underperform in organic search results so they can be prioritized for optimization?

### Current Files

- `01_first_look_and_discovery.ipynb` - Initial data discovery notebook
- `02_your_first_readable_model.ipynb` - Readable decision tree model notebook
- `research_question_framing.md` - Research question and project framing
- `data_contract.md` - Safe data usage and public reporting rules
- `baseline_plan.md` - Initial baseline strategy
- `project_notes.md` - Ongoing capstone notes
